Hi everyone, welcome to our portal OCPE. This is a guide for users on the features of OCPE

  • Login/Signup features

    • Signup:
      Cannot signup with existing username or password.
    • Login:
      Need to enter either email or username to login.Also doesn't allow a user of a type to access functionalites unique for user of the other type.Remember me feature to remember you if you do not logout before leaving the software.

  • Judge features

    • Create Problem:
      This section allows user of judge type to create a problem with all details such as name, title, description, sample input, sample output, time limit.
    • Modify Problem:
      This section allows user of judge type to modify the problem he/she created.The details that can be modified are name, title, description, sample input, sample output, time limit.

  • Contestant features

    • Practice:
      This section allows user of contestant type to practice the problems that have been set by the judges by submitting c codes for them.The software evaluates that and checks the output against the sample output and displays a verdict of the submission.
